Detailed Description

Klasa Prostopadloscian. Deklaracja klasy prostopadłościan. Zawiera ona elementy i metody potrzebne do wykonywania różnych operacji na prostopadłościanie.

Parameters
wspoldeklaracja prostopadłościanu jako 8 wektorów 3D. Wspol jest skrótem od współrzędna.

◆ dlugosc()

void Prostopadloscian::dlugosc ( )

Metoda sprawdzająca długości boków. Metoda sprawdzająca czy odpowiednie boki nowopowstałego prostopadłościanu są sobie równe. Wektory są obliczane ze wzorów matematycnych, tak samo ich długości. Porównywanie długości jest zrobione trochę nadgorliwie, ale to nie szkodzi. Wektory 0 - 3 to dolna podstawa, 4 - 7 to podstawa górna, a 8 - 11 to boki. Na koniec metoda wyświetla odpowiedni komunikat.

Parameters
WekXwektor o numerze X, dzięki któremu poprzez oblicznie jego długości liczone są długości boków prostopadłościanu
BokXbok prostopadłoscianu o numerze X
